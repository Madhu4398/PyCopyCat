Instructions

3 options to find plagiarism:

  1. File to File
  2. File to Files
  3. Folder Files

File to File

In File to File comparison, 2 python files are uploaded and checked for plagiarism.

File to Files

In File to Files comparison, a single python file and folder containing multiple files are uploaded and checked for plagiarism against each python files.
*Moredetails: In detail percentage of plagiarism by comparing file-file.

Folder Files

In Folder Files comparison, a folder containing multiple files are uploaded and checked for plagiarism among the files in the uploaded folder.
*Moredetails: In detail percentage of plagiarism by comparing file-file.

*WORKING OF PyCopyCat

The python files uploaded are stored onto the system ("file1 & file2"), then using AST python module the files are converted into AST format ("file1_dump & file2_dump"). Finally using the SequenceMatcher to get the ratio or percentage of plagiarism. The result page is diplayed with plagiarized percentage and both files show in a tabular view.
